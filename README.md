# TabZero

> Clean your tabs in 1 click.

TabZero is a minimal Chrome extension that organizes your messy tabs instantly.
No dashboards. No clutter. Just one action that works.

## ✨ What it does

* ⚡ **One-click cleanup**
  Group all open tabs instantly

---

## 🚀 How to use

1. Open TabZero
2. Click **Clean & Organize**
3. Done — your tabs are clean

---

## Technologies Used

- **React + Vite**
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion** (for smooth animations)
- **Lucide Icons**
- **Chrome Extension API**

## 📦 Installation (Development)

1. **Clone the repository**

```bash
git clone https://github.com/wrishab/tabzero.git
cd tabzero
```

2. **Install dependencies**

```bash
npm install
```

3. **Build the extension**

```bash
npm run build
```

4. **Load into Chrome**

* Open Chrome and go to: `chrome://extensions/`
* Enable **Developer mode** (top right)
* Click **Load unpacked**
* Select the `dist` folder
