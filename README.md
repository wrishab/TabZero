# TabZero: AI-Powered Tab Manager

TabZero is a modern Chrome extension designed to help you declutter your digital workspace. Instantly group tabs, remove duplicates, and save your current session for later.

## Features

- **Organize in 1 Click**: Uses smart categorization to group your tabs.
- **Rename & Recolor**: Customize your tab groups directly from the popup.
- **Save Sessions**: Save your entire workspace and restore it anytime.
- **Review Mode**: See all your active tabs and categories in a clean, visual interface.
- **Pinned Tabs Support**: Keep your most important tabs easily accessible.

## Installation for Development

1.  **Clone the repository**:
    ```bash
    git clone <your-repo-url>
    cd tabzero
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Build the project**:
    ```bash
    npm run build
    ```
4.  **Load in Chrome**:
    - Open Chrome and navigate to `chrome://extensions/`.
    - Enable **Developer mode** in the top right.
    - Click **Load unpacked**.
    - Select the `dist` folder generated by the build.

## Technologies Used

- **React + Vite**
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion** (for smooth animations)
- **Lucide Icons**
- **Chrome Extension API**

## Deployment

To export this app and upload it to GitHub:
1. Go to the **Settings** menu in the top right of the Google AI Studio environment.
2. Select **Export to GitHub**.
3. Link your repository and push the code!
